Module party_util

Utility functions for parties and alliances.

Functions

get_party_leader ()
Returns the party leader.

Returns:

    MobMetadata The mob data for the party leader, or nil if the player is not in a party.
is_party_leader (target_id)
Determines if a mob is the leader of the player's party.

Parameters:

  • target_id Mob id

Returns:

    Boolean True if the mob is the party leader and false otherwise.
get_party_members ()
Returns a list of party members.

Returns:

    list List of MobMetada for party members, or an empty list if the player is not in a party
get_party_member (target_id)
Returns a party member with the given id.

Parameters:

  • target_id Mob id

Returns:

    MobMetadata MobMetada for the party member, if a party member with the given id is in the party
is_party_member (target_id)
Determines if a mob is a member of the player's party.

Parameters:

  • target_id Mob id

Returns:

    Boolean True if the mob is the party leader and false otherwise.
party_claimed (target_id)
Determines if a mob is claimed by the player or a member of the player's party.

Parameters:

  • target_id Mob id

Returns:

    Boolean True if the mob is claimed by the player's party and false otherwise.
get_party_claimed_mobs (number)
Returns all nearby mobs that are party claimed.

Parameters:

  • number range_check Range check (optional)

Returns:

    list A list of indices of all mobs currently targeted by party members.
not_party_claimed (target_id)
Determines if a mob is claimed by a player outside of the player's party.

Parameters:

  • target_id Mob id

Returns:

    Boolean True if the mob is claimed by a player outside of the player's party.
party_targeted (target_id)
Determines if a mob is targeting the player or a member of the player's party.

Parameters:

  • target_id Mob id

Returns:

    Boolean True if the mob is targeting a player or a member of the player's party and false otherwise.
party_targets ()
Returns a list of indices of all mobs currently targeted by party members. The mobs do not have to be claimed as long as the target_index field of a party member's MobMetadata is not nil.

Returns:

    list A list of indices of all mobs currently targeted by party members.
get_buffs (target_id)
Returns a list of buff ids for a party member (see buffs.lua).

Parameters:

  • target_id Mob id

Returns:

    list A list of buff ids for a party member, or an empty list if the target is not in the player's party.
