Module Spell

Wrapper around a spell.

Functions

new (spell_name, job_abilities, job_names, target, consumable_name)
Default initializer for a new spell.

Parameters:

  • spell_name string Localized name of the spell
  • job_abilities list List of job abilities to use, if any
  • job_names list List of job short names that this spell applies to
  • target string Spell target (options: bt, p0...pn)
  • consumable_name string Name of consumable required to cast this spell (optional)

Returns:

    Spell A spell

Returns the names of the job abilities that should be used with this spell.

Returns:

    list Localized job ability names
get_job_names ()
Returns the list of jobs this spell applies to.

Returns:

    list List of job short names (e.g. BLU, RDM, WAR)
is_aoe ()
Returns whether or not this spell is AOE (e.g. Protectra).

Returns:

    Boolean True if the spell is AOE and false otherwise.
num_targets_required ()
The spell will not be cast unless at least this number of party members are in range (including the player).

Returns:

    number Number of targets
get_target ()
Returns the spell target.

Returns:

    string Spell target (e.g. bt, p1, p2)
get_range ()
Returns the range of the spell in yalms.

Returns:

    number Range of the spell (e.g. 18, 21, etc.)
get_consumable ()
Return the name of the consumable required to cast this spell.

Returns:

    string Name of the consumable (e.g. Shihei), or nil if none is required
