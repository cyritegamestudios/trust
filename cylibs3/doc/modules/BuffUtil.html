buff_util.is_buff_active (buff_id, player_buff_ids)
Determines if the player has a given buff active.

Parameters:

  • buff_id number Buff id (see buffs.lua)
  • player_buff_ids list List of active buff ids, included for performance reasons (optional)

Returns:

    Bool True if the buff is active and false otherwise.
