Module BattleStatTracker

Tracks a player's battle stats (e.g.

accuracy)

Functions

new (player_id)
Default initializer for a new battle stat stracker.

Parameters:

  • player_id number Player id

Returns:

    BattleStatTracker A battle stat tracker
destroy ()
Stops tracking the player's actions and disposes of all registered event handlers.
monitor ()
Starts tracking the player's actions. Note that it is necessary to call this before functions like getcurrenttarget(). Before this object is disposed of, stop_tracking() should be called.
reset ()
Resets the battle stats.
get_accuracy ()
Returns the player's accuracy since the battle stat tracker was last reset.

Returns:

    number Player's accuracy (between 0 and 100)
