Module Monster

Wrapper around monster metadata with additional functionality.

Functions

new (mob_id)
Default initializer for a new monster.

Parameters:

  • mob_id number Mob id

Returns:

    Monster A monster
destroy ()
Stops tracking the player's actions and disposes of all registered event handlers.
monitor ()
Starts tracking the monster's actions. Note that it is necessary to call this before functions like getcurrenttarget(). Before this object is disposed of, stop_tracking() should be called.
has_debuff (debuff_id)
Returns true if the monster has the given debuff.

Parameters:

  • debuff_id number Debuff id (see buffs.lua)

Returns:

    boolean True if the monster has the given debuff, false otherwise
get_current_target ()
Returns the current target of the monster.

Returns:

    MobMetadata Returns the full metadata for the monster's target, or nil if the monster isn't targeting anyone
get_mob ()
Returns the full metadata for the mob that this class wraps.

Returns:

    MobMetadata Mob metadata
