Module SongTracker

Tracks the songs on a list of players.

Functions

new (player_ids, dummy_songs, songs, pianissimo_songs)
Default initializer for a new song tracker.

Parameters:

  • player_ids number Player ids to track
  • dummy_songs List List of dummy songs (see spell.lua)
  • songs List List of songs (see spell.lua)
  • pianissimo_songs List List of songs to be sung with pianissimo after songs (see spell.lua)

Returns:

    SongTracker A song tracker
destroy ()
Stops tracking the player's actions and disposes of all registered event handlers.
monitor ()
Starts tracking the player's actions. Note that it is necessary to call this before songs will be tracked.

Call on every tic.
