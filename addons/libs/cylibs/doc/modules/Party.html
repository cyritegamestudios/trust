Module Party

Wrapper class around a party.

Functions

new ()
Default initializer for a Party.

Returns:

    Party A party
destroy ()
Stops tracking the player's actions and disposes of all registered event handlers.
monitor ()
Starts monitoring the player's actions. Note that it is necessary to call this before events will start being triggered. You should call destroy() to clean up listeners when you are done.
add_member (mob_id, name, main_job_id, sub_job_id, hpp, hp)
Adds a member to the party if they aren't in the party already, and updates the party member's main/sub job if they are.

Parameters:

  • mob_id number Party member id
  • name string Name of party member
  • main_job_id number Optional main job id
  • sub_job_id number Optional sub job id
  • hpp number Current hit point percentage
  • hp number Current hit points
get_party_members (include_self, range_check)
Returns all party members, optionally including the player.

Parameters:

  • include_self Boolean If true, the player will be returned
  • range_check number If non-nil, only returns party members that are < range_check yalms away from the player

Returns:

    list Party members
get_party_member (mob_id)
Returns the party member with the given mob id.

Parameters:

  • mob_id number Party member mob id

Returns:

    PartyMember Party member, or nil if none exists
get_party_member_named (mob_name)
Returns the party member with the given name.

Parameters:

  • mob_name string Party member mob name

Returns:

    PartyMember Party member, or nil if none exists
prune_party_members ()
Checks to see if each party member is still in the party. Removes invalid party members.
num_party_members ()
Returns the number of party members.

Returns:

    number Number of party members
set_assist_target (party_member)
Sets the assist target. The party's target will change to match the assist target.

Parameters:

  • party_member PartyMember Party member to assist
get_assist_target ()
Returns the assist target of the party.

Returns:

    PartyMember Assist target
