action_message_util.is_monster_defeated (message_id)
Determines if an action message indicates that a monster has been defeated.

Parameters:

  • message_id number Action message id (see action_messages.lua)

Returns:

    Bool True if the given message_id corresponds to a message indicating the monster has been defeated and false otherwise
