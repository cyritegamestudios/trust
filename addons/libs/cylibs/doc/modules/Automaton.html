Module Automaton

Base class for an automaton.

Functions

new (automaton_id, action_queue)
Default initializer for a new Automaton.

Parameters:

  • automaton_id number Mob id of the automaton
  • action_queue ActionQueue Action queue

Returns:

    Automaton An automaton
destroy ()
Stops tracking the player's actions and disposes of all registered event handlers.
monitor ()
Starts monitoring the automaton's actions. Note that it is necessary to call this before events will start being triggered. You should call destroy() to clean up listeners when you are done.
is_idle ()
Returns true if the automaton is currently not engaged.

Returns:

    Boolean True if the pet is idle, and false otherwise.
is_mage ()
Returns whether the pet is a mage (can cast magic).

Returns:

    Boolean True if the pet is a mage, and false otherwise.
get_provoke_recast ()
Returns the recast timer for Strobe.

Returns:

    number Recast in seconds, or MAX_NUM if a Strobe attachment is not equipped
get_flash_recast ()
Returns the recast timer for Flashbulb.

Returns:

    number Recast in seconds, or MAX_NUM if the Flashbulb attachment is not equipped
