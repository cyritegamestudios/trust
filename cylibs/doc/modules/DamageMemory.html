Module DamageMemory

Tracks the damage a player takes.

Functions

new (damage_threshold)
Default initializer for a new damage memory tracker.

Parameters:

  • damage_threshold number Only damage exceeding this amount will be recorded

Returns:

    DamageMemory A damage memory tracker
destroy ()
Stops tracking the player's actions and disposes of all registered event handlers.
monitor ()
Starts tracking damage taken. Note that it is necessary to call this before battle target tp moves will be tracked. Before this object is disposed of, destroy() should be called.
target_change (target_index)
Call when the target changes. Will begin tracking damage taken from tp moves initiated by the new target.

Parameters:

  • target_index number New target index
reset ()
Resets the damage memory.
record_damage (monster_ability_name, target_name, damage)
Records damage memory.

Parameters:

  • monster_ability_name string Name of monster ability (see monster_skills.lua)
  • target_name string Target of damage
  • damage number Damage taken
get_last_damage_taken (target_id)
Returns the damage taken for the last tp move to hit the given target.

Parameters:

  • target_id number Target id

Returns:

    Pair The pair monsterabilityname, damage
