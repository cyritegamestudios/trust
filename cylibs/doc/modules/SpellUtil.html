Module SpellUtil

Utility functions for spells.

Functions

spell_util.spell_id (spell_name)
Returns the spell id for the given localized spell name.

Parameters:

  • spell_name string Localized spell name

Returns:

    number Spell id (see spells.lua)
spell_util.spell_name (spell_id)
Returns the spell name for the given spell id.

Parameters:

  • spell_id number Spell id (see spells.lua)

Returns:

    string spell_name Localized spell name
spell_util.buff_id_for_spell (spell_id)
Returns the id for the buff associated with the given spell.

Parameters:

  • spell_id number Spell id (see spells.lua)

Returns:

    number Buff id (see buffs.lua)
spell_util.knows_spell (spell_id)
Returns whether the player knows a spell, either from their main job or sub job. Assumes all spells through the player's current level have been learned.

Parameters:

  • spell_id number Spell id (see spells.lua)

Returns:

    Boolean True if the player knows the given spell
spell_util.highest_spell_for_buff_id (buff_id, prefix)
Returns the highest tier spell that grants the given buff, e.g. given the buff_id for Refresh, returns Refresh III. Will only return spells the player can cast.

Parameters:

  • buff_id number Buff id (see buffs.lua)
  • prefix string Buff name prefix (optional) (see buffs.lua)

Returns:

    SpellMetadata Spell metadata (see spells.lua)
spell_util.spell_overwrites_spell (spell1_id, spell2_id)
Returns whether the first spell overwrites the second spell.

Parameters:

  • spell1_id number First spell id (see spells.lua)
  • spell2_id number Second spell id (see spells.lua)

Returns:

    Boolean True if the first spell overwrites the second spell
spell_util.can_cast_spell (spell_id)
Returns whether the player can cast a spell.

Parameters:

  • spell_id number Spell id (see spells.lua)

Returns:

    Boolean True if the player knows the spell and its recast timer is up.
spell_util.get_spell_recast (spell_id)
Returns the spell recast for the given spell. Note that this will return 0 if a player has learned a spell but does not have access to it on their current job.

Parameters:

  • spell_id number Spell id (see spells.lua)

Returns:

    number Recast time (in seconds)
spell_util.can_cast_spells ()
Returns whether the player is able to cast spells (e.g. if they are silenced).

Returns:

    Boolean True if the player can cast spells and false otherwise
