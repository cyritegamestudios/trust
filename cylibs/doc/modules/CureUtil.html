Module CureUtil

Utility functions for healing magic.

Functions

cure_util.spell_id_for_debuff_id (debuff_id)
Determines the spell that can remove a debuff.

Parameters:

  • debuff_id number Debuff id (see buffs.lua)

Returns:

    number Spell id for the spell that can remove the debuff, or nil if none exists
cure_util.get_cure_priority (hpp, is_trust, is_aoe)
Determines the priority of a cure.

Parameters:

  • hpp number Cure target's current hit point percentage
  • is_trust Boolean Whether the cure is being applied to a trust
  • is_aoe Boolean Whether the cure is AOE

Returns:

    ActionPriority Action priority, to be used with an action
